#LINUX ARCHITECTURE NOTES#

1.Core components of linux: 
   (1)Kernal:Main part of linux. it manage the hardware like CPU,memory, Device.
   (2)User space: Where the applications run.
   (3)systemd: it is the init system that starts and manages the services.
   
2.Processes:
    A process is the running program. 
    (1)processes are created using fork()
    (2)New program run using exec()
    (3)One process can create another process.
    
3.systemd:
    (1)start system services at boot
    (2)manages backround services
    (3)helps in system logging
    
4.shell: interface between the user and kernel
